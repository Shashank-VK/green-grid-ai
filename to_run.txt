Run them separately right now. There isn’t a root “start both” script in this repo yet.

Use two terminals from ev-grid-scanner:

Backend
cd backend
.\venv\Scripts\activate
python -m uvicorn app.main:app --reload --port 8000
Frontend
cd frontend
npm run dev
Then open:

Frontend: http://localhost:3000
Backend health: http://localhost:8000/api/health
If npm install hasn’t been run yet in frontend, do that once first:

cd frontend
npm install
